# EliteShine Cleaning Agency

A professional cleaning agency website built with HTML, Tailwind CSS, and Font Awesome.

## Project Structure

```
EliteShine-Cleaning-Agency/
├── elite.html      # Home page
├── contact.html    # Contact page
└── README.md      # This file
```

## Features

- **Responsive Design** — Mobile-friendly layout using Tailwind CSS
- **Navigation** — Sticky navbar with desktop and mobile hamburger menu
- **Hero Section** — Eye-catching landing section with call-to-action
- **Services** — Professional cleaning services showcase
- **About Section** — Company information and value proposition
- **Contact Form** — Functional contact page for customer inquiries
- **Icons** — Font Awesome integration for visual elements

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome 6.4.0 (via CDN)
- JavaScript (vanilla)

## Getting Started

1. Clone or download the repository
2. Open `elite.html` in your web browser
3. Navigate between pages using the navigation menu

## Pages

| Page           | Description                                       |
| -------------- | ------------------------------------------------- |
| `elite.html`   | Home page with hero, services, and about sections |
| `contact.html` | Contact page with form and company information    |

## Customization

- Modify colors in Tailwind classes (e.g., `bg-blue-600`)
- Update content in HTML files
- Add more pages as needed
